Commands

[parameter] : Required Parameter
{parameter} : Optional Parameter


/play [song]

Use this to play a song. You must be in a voice channel to use this. The [song] parameter can be the name of a song, link to a song, or link to a playlist. The bot supports songs from Spotify, YouTube, Soundcloud, Vimeo, Reverbnation, and Facebook.


/queue {page}

Use this to see your server's queue. To view a specific page (if there are multiple) use the page number.


/stop

Use this to make the bot stop playing music, leave the VC, and destroy the queue.


/shuffle

Use this to shuffle the order of the songs in the queue.


/nowplaying

Use this to get some information about the song that is currently playing.


/pause

Use this to pause the song that is playing.


/resume

Use this to resume the paused song.


/skip

Use this to skip the current song and play the next in queue.


/skipto [queue number]

Use this to skip the current song and play a specific song in queue. To get the [queue number] use the /queue command and look at the number next to the song you want.


/clear

Use this to clear all the songs out of the queue.


/loop

Use this to loop the current song.


/loopqueue

Use this to loop the queue.


/stoploop

Use this to break the loop, works if its the song loop or the queue loop.


/lyrics

Use this to get the lyrics for the song that is currently playing. This normally only works if the source of the song is Spotify, Soundcloud, or Reverbnation.
